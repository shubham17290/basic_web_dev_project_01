# my-javadoc-portal

This folder contains a JavaDoc-style documentation portal for a small Java project.

## Files
- `index.html` — entry page that redirects to `package-summary.html`.
- `package-summary.html` — package overview with links to class documentation.
- `allclasses.html` — class index with links to `Love.html` and `hum.html`.
- `Love.html` — class documentation page for `Love`.
- `hum.html` — class documentation page for `hum`.
- `constant-values.html` — constant field documentation page.
- `script.js` — shared documentation script with utility and navigation helpers.
- `stylesheet.css` — shared stylesheet referenced by pages.
- `first.java` — Java source which compiles successfully.
- `package.json` — Node.js backend manifest.
- `server.js` — simple Express server to host the portal locally.

## How to use
1. Open `index.html` in a browser.
2. The page will redirect to `package-summary.html`.
3. Use the top navigation links to jump between pages.

## Run locally with backend
In the portal folder:

```bash
npm install
npm start
```

Then open:

```bash
http://localhost:3000
```

## Notes
- Local references are valid and all direct links have been updated.
- The portal supports a polished class directory, package overview, and constant reference page.
- The backend exposes `http://localhost:3000/api/status` for health checks.
